Welcome to VM Service Slide

This dashboard provides documentation, capabilities, and API references for the VM Service Slide repository.

Quick Start

To get started with the API, you will need to authenticate and obtain a Bearer token.

curl -X GET "https://api.yourdomain.com/health"

System Capabilities

The VM Service Slide provides the following core functionalities:

  • Slide Generation: Programmatically generate presentation slides from raw data.
  • Template Management: Store, retrieve, and version custom slide templates.
  • Exporting: Export compiled slides to PDF, PPTX, and HTML formats.

Slides API

Use these endpoints to manage your slide decks.

GET /api/v1/slides

Retrieves a paginated list of all slide decks associated with the authenticated user.

POST /api/v1/slides/generate

Creates a new slide deck based on the provided JSON payload.

{
  "template_id": "temp_123",
  "title": "Q3 Report",
  "content": [...]
}
DELETE /api/v1/slides/{id}

Permanently removes a slide deck from the system.

Authentication

All API requests require a valid JWT token passed in the Authorization header.

Authorization: Bearer YOUR_ACCESS_TOKEN
